\# Task Manager (Node.js + Express + MongoDB)



A simple Task Manager REST API built with Node.js, Express, and MongoDB.  

It supports full CRUD operations: Create, Read, Update, Delete tasks.



\## Features

\- Create tasks (POST)

\- View all tasks (GET)

\- Update task status (PUT)

\- Delete tasks (DELETE)

\- Data stored in MongoDB



\## Tech Stack

\- Node.js

\- Express.js

\- MongoDB (Mongoose)

\- Postman for testing



\## API Routes



| Method | Route           | Description |

|--------|----------------|-------------|

| GET    | /tasks         | Get all tasks |

| POST   | /tasks         | Create new task |

| PUT    | /tasks/:id     | Update task completed status |

| DELETE | /tasks/:id     | Delete task |
