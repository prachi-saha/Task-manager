\# 📝 Task Manager API



A simple backend application to manage tasks using REST APIs.



\---



\## 🚀 Features

\- Create a task

\- View all tasks

\- Update a task

\- Delete a task



\---



\## 🛠 Tech Stack

\- Node.js

\- Express.js

\- MongoDB

\- Postman (for testing APIs)



\---



\## 📦 Installation



```bash

npm install

node server.js
